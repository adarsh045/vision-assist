## Vision Assist

Core module providing object detection and memory functionality for the main application.

#### The module will implement the following features:

- Object Detection: Utilize computer vision techniques to identify and classify objects within images or video streams.
- Memory Storage: Store detected objects and their attributes in a structured format for future reference.
- Retrieval System: Implement a mechanism to query and retrieve stored objects based on specific criteria.
- Speech to Text : Implement speech recognition to convert spoken commands into text for processing user request. 
- Text to Speech : Implement text-to-speech functionality to provide audio feedback to the user based on system responses.
