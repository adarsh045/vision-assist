## SAP Project

### Title : A Smart Assistive System for Detecting Forgotten Belongings and Providing Speech-Based Guidance to the Blind

#### Summary:

This project aims to develop a smart assistive system designed to help visually impaired individuals by detecting forgotten belongings and providing speech-based guidance. The system utilizes advanced technologies such as computer vision, machine learning, and natural language processing to identify items that users may have left behind and offer verbal reminders of the exact place where the user had left it based on the enivronment captured.
